# Sistema de Notas em Python

Projeto desenvolvido para praticar conceitos básicos de Python.

## Funcionalidades

- Cadastro do nome do aluno
- Registro de três notas
- Cálculo automático da média
- Verificação da situação do aluno

## Tecnologias Utilizadas

- Python

## Objetivo

Praticar lógica de programação, estruturas condicionais e manipulação de dados.
